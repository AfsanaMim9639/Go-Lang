
## 🧠 Compiler

**Definition:**
A compiler is a **translator** for our code.
It converts human-readable Go source code into machine code that the computer can execute.

---

## ⚙️ Runtime

**Definition:**
The runtime is responsible for managing the **execution** of our program — including:

* **Memory allocation**
* **Garbage collection**
* **Execution flow**
